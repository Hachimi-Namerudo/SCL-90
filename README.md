# SCL-90
SCL-90（symptom checklist 90），90项症状清单，是一份在当代被广泛运用的心理健康测试量表。

这是借助Qt 6.9.2开发的一个心理健康状况自评量表。

A self-assessment scale for mental health status made by Qt 6.9.2

目前可以正常使用。要使用程序，请将下载的.zip解压，随后运行SCL-90.exe。

Now it functions normally.To run the program,please unzip the .zip file and run SCL-90.exe.
